# ADR 001: Technology Stack Selection

## Status: [![Project Status: Proposed](https://img.shields.io/badge/Technology%20Stack%20Selection%20Status-Proposed-yellow)](https://github.com/)


## Context
The project requires a web application for facilitating group consultations. We need to select a tech stack that satisfies the brief's constraints while allowing for a good user experience.

## Decision
1. **Runtime**: Node.js
2. **Framework**: Express.js
3. **Frontend**: Bootstrap 
4. **Database**: MongoDB
5. **Hosting**: Render 

## Benfits
Using these technologies ensures compliance with the technical requirements (Section 3.3). Node/Express allows for rapid iterative development required by the 4-sprint cycle.
